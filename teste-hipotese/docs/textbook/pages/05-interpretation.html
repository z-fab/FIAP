Apostila Interativa

Interpretação de Resultados

Um p-valor baixo não é o fim da análise — é o começo. Nesta seção, você aprende a traduzir números em decisões de negócio.

5.1 Quando p < α: Resultado Significativo

Rejeitar H0 significa que a diferença observada é improvável sob a hipótese nula. Porém, significância estatística não implica relevância prática. Antes de celebrar, faça três perguntas:

  1. A diferença é grande o suficiente? Um p=0.001 com diferença de 0.002 em AUC é estatisticamente significativo mas irrelevante na prática. A magnitude importa mais que o p-valor.
  2. É consistente entre métricas? Se o modelo A é melhor em F1 mas pior em Recall, a "superioridade" depende do que você prioriza. Verifique múltiplas métricas antes de decidir.
  3. O custo compensa? Um ganho de 0.5% em acurácia pode não justificar um modelo 10x mais lento em produção, 3x mais caro em GPU, ou impossível de explicar ao regulador.

Para organizar essas perguntas, podemos mapear qualquer resultado em um quadrante de decisão baseado em duas dimensões: o p-valor e o tamanho do efeito (diferença prática).

5.2 Significância Estatística vs. Significância Prática

Este é o erro mais comum: confundir significância estatística com significância prática. Com amostras grandes (muitos folds ou repetições), o teste ganha poder para detectar diferenças minúsculas — que podem ser reais mas completamente irrelevantes.

Armadilha Clássica

p=0.001, diferença AUC=0.002. O teste detectou uma diferença real, mas tão pequena que nenhuma decisão de negócio deveria se basear nela. Sempre reporte a magnitude da diferença junto com o p-valor.

Balança: Estatística vs. Prática
Uma diferença pode ser estatisticamente significativa sem ser praticamente relevante
Exemplo de Boa Interpretação

"O teste t pareado indica diferença significativa (p=0.003). A diferença média de 3.2 pontos percentuais em F1-score (IC 95%: [1.8, 4.6]) é relevante para o caso de uso, onde cada ponto percentual representa ~R$50k em fraudes detectadas."

Intervalos de Confiança

O exemplo acima menciona "IC 95%: [1.8, 4.6]". Mas o que exatamente é esse intervalo, como calculá-lo, e por que ele é tão importante quanto o p-valor?

Fórmula do Intervalo de Confiança

Para diferenças pareadas com K folds:

$$IC = \bar{d} \pm t_{\alpha/2,\; K-1} \times SE \qquad \text{onde } SE = \frac{SD(d)}{\sqrt{K}}$$

Onde d̄ é a média das diferenças, SD(d) é o desvio padrão, e K é o número de folds.

O intervalo de confiança de 95% significa: "Temos 95% de confiança de que a verdadeira diferença de desempenho entre os modelos está entre [limite inferior, limite superior]".

Python
import numpy as np
from scipy import stats

differences = scores_a - scores_b
mean_diff = np.mean(differences)
se = stats.sem(differences)
ci = stats.t.interval(0.95, df=len(differences)-1, loc=mean_diff, scale=se)
print(f"Diferença média: {mean_diff:.4f}")
print(f"IC 95%: [{ci[0]:.4f}, {ci[1]:.4f}]")
Conexão com o p-valor

Se o intervalo de confiança de 95% inclui zero, isso é consistente com p ≥ 0.05 — não há evidência de diferença significativa. Se o IC não inclui zero, a diferença é significativa ao nível de 5%.

5.3 Quando p ≥ α: Ausência de Evidência

O que p ≥ α NÃO significa

"Os modelos são iguais" — ERRADO. Não rejeitar H0 não significa que H0 é verdadeira. Significa apenas que não há evidência suficiente para concluir que são diferentes. Ausência de evidência não é evidência de ausência.

Existem três razões principais pelas quais um teste pode não rejeitar H0:

1. Poder estatístico insuficiente

Com poucos folds (ex: K=5), o teste pode simplesmente não ter sensibilidade para detectar diferenças reais. Aumente K (10 ou 30) ou use repeated K-fold para ganhar poder.

2. A diferença realmente não existe

Os modelos podem ter desempenho genuinamente equivalente. Nesse caso, prefira o modelo mais simples: menos parâmetros, mais rápido, mais interpretável.

3. Alta variabilidade nos dados

Se os folds produzem resultados muito diferentes entre si (alto desvio-padrão), o "ruído" mascara o "sinal". Considere estratificação ou aumento de dados.

5.5 Quando Resultados Divergem

É comum que um paired t-test entre dois modelos dê significativo, mas o Nemenyi post-hoc mostre o mesmo par como não significativo. Isso não é um erro — são perguntas diferentes:

Paired t-testNemenyi
PerguntaEsses 2 modelos específicos diferem?Quais pares diferem entre TODOS?
CorreçãoNenhuma (1 comparação)Corrige para $C(k,2)$ comparações
Quando usarComparação única, pré-planejadaExploração entre múltiplos modelos
PoderMaior (limiar mais baixo)Menor (limiar mais alto)
Em qual confiar?

Se você planejou comparar exatamente 2 modelos desde o início → t-test (comparação pré-planejada, sem correção).

Se você está explorando entre vários modelos → Nemenyi (controla falsos positivos globais).

Na dúvida, o Nemenyi é mais conservador.
